### UbiGamesBackupTool
### 育碧游戏存档备份工具

#### 说明

- Uplay在注册表中的位置<br>
计算机\HKEY_LOCAL_MACHINE\SOFTWARE\Classes\uplay\Shell\Open\Command

- 用户的UID与用户名相对应文件<br>
%USERPROFILE%\AppData\Local\Ubisoft Game Launcher\users.dat

#### 现有特性
- 自动探测路径

#### 计划特性
- 支持检测多个Uplay用户
- 支持备份指定游戏
